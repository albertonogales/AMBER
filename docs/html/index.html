🔶 AMBER

Autoassociative Map Builder for tEmporal Representations — a Python library for Self-Organizing Maps with temporal context, seven distance metrics, and integrated visualisation.

Python ≥ 3.9 372 Tests Passing GPL-3.0 License v2.2.0

Features

🗺️

Self-Organizing Maps

Full SOM training pipeline: Vesanto size heuristic, four weight-init strategies, cosine learning-rate decay, and quality metrics (QE, TE).

📏

Seven Distance Metrics

Euclidean, Manhattan, Chebyshev, Cosine, Correlation, DTW, and Cross-correlation — the only Python SOM library supporting DTW and XCorr.

⏱️

Temporal / Recurrent SOM

TemporalMap adds a context vector (Voegtlin 2002 RSOM) that accumulates sequence history, enabling regime-change detection in time series.

🔬

Feature Extractor

20+ domain-specific features for biosignals and audio: EEG band powers, Hjorth parameters, spectral entropy, MFCCs, and more.

📊

Built-in Visualisation

Heat map, U-matrix, elevation map, codebook vectors, temporal trajectory, transition matrix, and dwell-time map — all one-liners.

🧪

Fully Tested

372 pytest tests, 99% coverage. All tests run on synthetic data — no external files or internet access required.

Workflow

1
Train
Map(data, period) — choose size, distance, normalisation and weight init. The Vesanto heuristic sets map size automatically.
▼
2
Classify
Classification(som, data) — BMU assignment, activation map, U-matrix, quantisation error, topological error.
▼
3
Temporal analysis
TemporalMap + TemporalAnalysis — transition matrix, dwell times, trajectory, stability metrics.
▼
4
Visualise
Visualization.heat_map(cls), umatrix(cls), trajectory(tcls, ta), …
▼
5
Save / Load
som.save_classifier('model') / Map.load_classifier('model') — JSON serialisation, no pickle.

Quick Start

pip install amber-som
import numpy as np
import AMBER

data = np.random.rand(300, 10)   # 300 samples, 10 features

# Train
som = AMBER.Map(data=data, period=200)
print(f"Map size: {som.map_size}×{som.map_size}")   # auto-sized

# Classify
cls = AMBER.Classification(som, data)
print(f"Quantisation error : {cls.quantization_error:.4f}")
print(f"Topological error  : {cls.topological_error:.4f}")

# Visualise
AMBER.Visualization.heat_map(cls, colorscale='Reds')
AMBER.Visualization.umatrix(cls)
